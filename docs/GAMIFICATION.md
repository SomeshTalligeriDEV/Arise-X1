# Gamification Design

## XP System

- Every logged meal: +50 XP
- Quest completion: +100–300 XP (by difficulty)
- Walk spot claimed: +25–75 XP
- Daily streak bonus: +20 XP/day
- Hydration goal met: +30 XP

## Leveling

Levels follow a quadratic curve. Level 1 requires 100 XP; each subsequent
level requires ~20% more than the previous. This keeps early progress fast
and long-term engagement meaningful.

## Quests

| Type | Reset | Example |
|------|-------|---------|
| Daily | Midnight | "Log 3 meals today" |
| Weekly | Monday | "Walk 10,000 steps this week" |

Quest categories: `nutrition`, `movement`, `hydration`, `mindfulness`.

## Coins

Coins are a secondary currency earned alongside XP. Planned use:
- Unlock premium meal plan filters
- Cosmetic avatar items
- Community challenge entry fees

## Behavioral Pillars

1. **Data-driven personalization** — recommendations grounded in your history
2. **Contextual intelligence** — time, mood, location shape every suggestion
3. **Behavioral change loops** — Tiny Habits + variable rewards, not addiction
