# ARISE-X1 Design System

## Color Palette

The app uses a dark-first palette with neon accents.

| Token | Value | Usage |
|-------|-------|-------|
| `--primary` | `#7c3aed` (violet) | XP, levels, CTAs |
| `--accent` | `#06b6d4` (cyan) | Secondary highlights |
| `--background` | `#0f0a1e` | Page background |
| `--foreground` | `#f8fafc` | Body text |
| `--muted-foreground` | `#94a3b8` | Secondary text |

## Typography

| Font | Usage |
|------|-------|
| Orbitron | Headings, level numbers |
| Press Start 2P | Pixel-art decorative text |
| VT323 | Terminal / retro UI elements |
| System sans-serif | Body copy |

## Glass Morphism

Three glass utility classes:
- `.glass` — subtle frosted panel
- `.glass-strong` — more opaque, used for featured cards
- `.glow-primary` — violet drop shadow for CTAs

## Animation

- `.animate-float-slow` — gentle vertical float for tooltip pills
- `.animate-pulse-ring` — expanding ring for map markers
- `.bg-aurora` — animated gradient background
