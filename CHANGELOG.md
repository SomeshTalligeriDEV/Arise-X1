# Changelog

All notable changes to ARISE-X1 are documented here.

## [0.1.0] — 2025-01-17

### Added
- Landing page with hero, stats, meal showcase, persona grid and pricing
- Dashboard with XP bar, streak counter, mood selector and hydration tracker
- Quests system with daily and weekly challenges
- Explore page with live walking map (Pokémon GO–style)
- Leaderboard with community rankings
- Plans page with subscription tiers
- Recommend page with ARIA questionnaire
- Voice input page with Sarvam integration
- ARIA floating AI coach with contextual nudges
- Full shadcn/ui component library
- UserStatsContext for global XP, coins and streak state
- Dark-first design with aurora gradients and glass morphism

### Documentation
- Added ARCHITECTURE.md with routing and state overview
- Added DESIGN_SYSTEM.md with colour tokens and typography
- Added GAMIFICATION.md with XP, leveling and quest design
- Added CONTRIBUTING.md with setup and commit style guide
